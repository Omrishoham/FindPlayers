-Andrid app wrote in Java.

-This app offers a framework that mediates between people who want to play with each other in several types of activities and games.

-User will be able to register for the app and through a google map view,
to see the activities closest to his current location and decide whether to join them.

-In addition each user has an activity creation option and can define the activity type, time, date, number of participants and more.

-Server side and SQL used with Firebase.
